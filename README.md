## Requisitos

- **SDL2 2.0.12+** ([Official Site](https://www.libsdl.org/))
- **FreeType 2.13.0+** ([Documentación](https://freetype.org/))
- **GLEW 2.1.0+** ([Repositorio](https://github.com/nigels-com/glew))
- Compilador MSVC o compatible C11
